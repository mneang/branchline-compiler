# Product Thesis

## Target User

Solo and small-team creators building branching visual novels,
interactive comics, and narrative games with generated media.

## Critical Pain

A small story edit can silently leave downstream voices, captions,
scene renders, previews, or entire reachable branches outdated.

## Core Insight

Generating media is not the hardest problem.

The harder production problem is knowing exactly which assets became
stale, rebuilding only those assets, and proving that every playable
path is current before release.

## Mission

Detect source changes, calculate the minimum media rebuild plan,
request approval, rebuild stale assets through Genblaze, reuse valid
assets from Backblaze B2, test every reachable path, and block release
if stale or missing media remains.

## Magic Moment

The creator changes one shared story line.

Both affected branches become visibly stale.

Only the dependent assets rebuild.

All reachable paths are verified.

The new release is published with zero stale assets.
